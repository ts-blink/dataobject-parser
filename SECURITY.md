# Security Policy

## Supported Versions

Currently, only the most recent version 1.2.x supports security updates.  

| Version | Supported          |
| ------- | ------------------ |
| 1.2.22  | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you see a security issue feel free to open a PR and document the issue.
